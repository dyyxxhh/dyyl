## 语法约定

- **参数分隔**：同级参数用逗号 `,` 分隔
- **占位符**：`_` 表示空参数，`empty` 等同 `_`，用于跳过某个参数位
- **括号定界**：用 `()` 包裹内层命令的参数，消除嵌套歧义
- **注释**：`#` 开始行注释，支持行首注释和行内注释
- **命令行**：每条命令必须在一行内完成，不支持续行
- **数值字面量**：源码可直接写整数、分数、根式和常量（如 `3`、`1/3`、`√2`、`π`）
- **可选引号**：字符串参数无需引号；若参数含逗号、空格等特殊字符，可用双引号 `"` 包裹（如 `"hello, world"`）
- **反斜杠转义**：参数中含特殊字符时可用 `\` 转义（如 `hello\, world`），引号内的 `\n` `\t` `\\` `\"` 也按标准转义处理
- **裸词参数**：不含特殊字符的普通参数直接书写，无需引号
- **字符串内逗号**：单引号/裸词内逗号不分割参数；仅裸露、未被转义且不在引号内的逗号才作为参数分隔符

```
# 右值贪心：math.add 及其后参数全部是 set 的右值
set $i, math.add $i, 1
# 等价写法：用 () 显式定界
set $i, math.add($i, 1)
# 左侧歧义时用占位符 _ 消歧
logic.or math.add(a, b), _, c
# 字符串引号：以下两个命令等价
io.out hello world
io.out "hello world"
# 反斜杠转义：参数内逗号不被切分
io.out hello\, world
# 转义序列：引号内 \n 表示换行
io.out "line one\nline two"
```

## 变量类型

| 命令 | 参数 | 说明 |
|------|------|------|
| `create.str` | `变量名` | 创建字符串变量（初值 empty） |
| `create.num` | `变量名` | 创建数值变量（初值 0，统一类型，见下方说明） |
| `set` | `$变量名, 右值表达式` | 赋值：右值可为字面量、`$变量名` 取值、或命令调用 |
| `$变量名` | — | 获取变量值 |

### `create.num` — 统一数值类型

`create.num` 内部以符号表达式存储，自动判断形式，全程无损计算，输出时还原原始表达。

| 输入 | 内部存储 | 输出 |
|------|----------|------|
| `3` | 整数 `3` | `3` |
| `1/3` | 分数 `(1,3)` | `⅓` |
| `π` | 符号 `π` | `π` |
| `√2` | 符号 `√2` | `√2` |
| `3/2` | 分数 `(3,2)` | `1½` |
| `5/3` | 分数 `(5,3)` | `1⅔` |
| `√2/2` | 分数 `(√2,2)` | `(√2)/2` |
| `√(3+1)/2` | 分数 `(√(3+1),2)` | `√(3+1)/2` |
| `1/3 + π` | 表达式 `(1,3) + π` | `⅓ + π` |
| `1/3 × 3` | 计算得 `(3,3)` → 约分 `1` | `1` |

> **输出优先级**：带分数 > 假分数 > 无理数表达式 > 小数近似值。
> 能写成带分数就写带分数（`5/3` → `1⅔`），因为带分数精确无损，而 `1.666...` 是无限小数无法精确表示。无理数无法写成带分数时保留普通分数（`(√2)/2`）。只在显式调用 `math.approx` 时才降级为小数。
>
> **括号规则**：`√` 后跟单个数/符号时不加括号（`√2`）；后跟表达式时加括号（`√(3+1)`）。根式作为分子时整体加括号（`(√2)/2`），避免歧义。

## 逻辑控制

| 命令 | 参数 | 说明 |
|------|------|------|
| `logic.un` | `x` | 非（NOT） |
| `logic.and` | `x, y` | 与（AND） |
| `logic.or` | `x, y` | 或（OR） |
| `logic.same` | `x, y` | 判断 x 与 y 是否相等；类型不同直接不等 |
| `logic.not.same` | `x, y` | 判断 x 与 y 是否不等 |
| `logic.more` | `x, y` | x > y |
| `logic.less` | `x, y` | x < y |
| `logic.more.same` | `x, y` | x ≥ y |
| `logic.less.same` | `x, y` | x ≤ y |
| `logic.max` | `x, y` | 取 x 和 y 中较大值 |
| `logic.min` | `x, y` | 取 x 和 y 中较小值 |
| `logic.between` | `x, 下限, 上限` | 判断 x 是否在 [下限, 上限] 区间内 |
| `logic.clamp` | `x, 下限, 上限` | 将 x 限制在 [下限, 上限] 内 |
| `logic.is.num` | `x` | 判断 x 是否为数值类型 |
| `logic.is.str` | `x` | 判断 x 是否为字符串类型 |
| `logic.is.empty` | `x` | 判断 x 是否为空（空字符串/0/空字典/空列表） |
| `logic.if` | `条件, 行数` | 满足条件时执行 |
| `logic.else` | `条件, 行数` | 上一个 if 条件为假且本条件为真时执行（链式 elif 语义，仅链接上一个 if） |
| `logic.while` | `条件, 行数` | 满足条件时循环；返回实际执行次数 |
| `logic.for` | `次数, 行数` | 循环指定次数；返回实际执行次数 |

> **返回值与条件**：逻辑判断返回数值 `1`（真）或 `0`（假）；`logic.if` / `logic.else` 返回 `1` 或 `0`；`logic.while` / `logic.for` 返回实际执行次数。条件中非 `0` 为真，`0` 为假。
> **嵌套行数**：控制流可以嵌套。内层控制流自身 1 行 + 内层块体行数都计入外层块体行数；外层声明的行数不足以覆盖内层完整块体时返回哨兵。
> **作用域**：变量为全局作用域，块体内 `create.*` 的变量在块体外仍可访问。

## 字典 / 键值对

| 命令 | 参数 | 说明 |
|------|------|------|
| `dict.create` | `变量名` | 创建空字典 |
| `dict.set` | `字典, 键, 值` | 设置键值对（键可为任意类型） |
| `dict.get` | `字典, 键` | 获取键对应的值；缺键返回 -1 |
| `dict.has` | `字典, 键` | 判断键是否存在 |
| `dict.del` | `字典, 键` | 删除键值对 |
| `dict.keys` | `字典` | 获取所有键（列表） |
| `dict.vals` | `字典` | 获取所有值（列表） |
| `dict.len` | `字典` | 获取键值对数量 |

```
# 统计词频
dict.create freq
dict.set freq, "hello", 3
dict.set freq, "world", 1
io.out dict.get(freq, "hello")         # 3

# 遍历所有键（list.len 索引取值）
create.num i
set $i, 0
logic.while logic.less($i, list.len(dict.keys(freq))), 2
  io.out list.get(dict.keys(freq), $i)
  set $i, math.add($i, 1)

# 缓存计算结果
dict.create cache
logic.if logic.un(dict.has(cache, key)), 2
  dict.set cache, key, math.sqrt(2)
io.out dict.get(cache, key)            # (√2)
```

## CLI 命令

`cli.*` 命令族提供对脚本文件名之后命令行参数的只读访问。参数由解释器(`main.rs`)注入运行时环境;通过库 API 调用且未显式注入时,参数为空。

| 命令 | 参数 | 返回 | 说明 |
|------|------|------|------|
| `cli.args` | — | `List<Str>` | 文件名之后的所有参数,按原顺序。无参时返回空列表 |
| `cli.count` | — | `Num` | 参数个数。无参时返回 0 |
| `cli.get` | 一个非负整数(0-based) | `Str` 或 `Num(-1)` | 按下标取参数。越界或负数返回 `Num(-1)` |
| `cli.has` | 一个字符串(flag 名) | `Num(1)` 或 `Num(0)` | 精确匹配 flag,或 `--flag=...` 形式(`--flag` 部分相等即算存在)。不做前缀匹配 |
| `cli.value` | 一个字符串(flag 名) | `Str` 或 `Empty` | 取 `--flag value`(空格分隔)或 `--flag=value`(等号)的值。找不到或 flag 后无值返回 `Empty`。多次出现返回第一个值 |
| `cli.script_name` | — | `Str` | 脚本文件名的 basename(如 `/home/user/a.dyyl` → `a.dyyl`) |
